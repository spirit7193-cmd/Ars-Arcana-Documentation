Item Incants directly pull an item from an inventory, returning it as a Cluster. With such a specific definition, it makes sense why there is only two most will learn by default.

In this section, you will learn their names and functions.

**BURITE**

**SYNTAX**

_Akashic -> Entity/Block, Number_

_Burite requires 2 Clusters to function properly._

_The Entity/Block Cluster serves as the inventory that the sought after item is in._

_The Number Cluster serves as the slot to look in. As with most Akashic things, slots begin counting at 0._

_Akashic <- Item/Null_

_Burite is the incant that represents "find item". It is self explanatory. Provided with an inventory, it returns the item in the specified slot. If there is nothing in the slot, a Null Cluster is returned instead of the promised Item Cluster. Whatever is returned is added to the top of the Akashic Stack._

**Example Spells:**

ETE 0 BURITE -> Returns the Item in the caster's first hotbar slot, otherwise returns Null if nothing is there.

**KALBURITE**

**SYNTAX**

_Akashic -> Vector_

_Kalburite requires 1 Cluster to function properly._

_The Vector Cluster serves as the position of the item frame._

_Akashic <- Frame Item/Null_

_Kalburite is the incant that represents "get item in item frame". It is self explanatory. Provided with a Vector that points to an item frame, it returns the item inside it. If there is nothing in the frame, a Frame Item Cluster is still returned but it will be the frame itself. If there is no frame, Null is returned. Whatever is returned is added to the top of the Akashic Stack._

**BURFANITE**

**SYNTAX**

_Akashic -> Item_

_Burfanite requires 1 Cluster to function properly._

_The Item Cluster serves as the item the resulting Spell List is pulled from._

_Akashic <- List (Incants)_

_Burfanite is the incant that represents "get spell from item". It is self explanatory. Provided with an Item (usually a Spell Page or a Spell Book), it draws the saved Spell List from it, adding it to the top of the Akashic. The List that is returned is added to the top of the Akashic Stack._
