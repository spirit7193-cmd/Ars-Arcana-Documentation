A good foundation in arcanic is paramount to proper spellwriting. In this section, there are incants that will prove necessary for any form of spell to either work properly or be properly monitored and fixed, should any issue in magical grammar arise.

**ETE**

**SYNTAX**

_Akashic -> No Clusters_

_Ete requires no clusters to function. This is a major reason why it can be used to begin spells._

_Akashic <- Entity_

_Ete is the incant that signifies “I”. It gathers the caster of the spell. That is, it pushes an Entity Cluster into the Akashic Stack._

**Example Spells:**

ETE DEBUG -> Allows the caster to see the Akashic and view their own cluster.

ETE GEA FUS -> Conjures a magical light at the caster’s feet.

ETE BIEGEA FUS -> Conjures a magical light at the caster’s head.

**DEBUG**

**SYNTAX**

_Akashic -> No Clusters_

_Debug does not need any clusters to function. However, it is good to have a few in the Akashic, else whatever information gained will return Empty._

_Akashic <- No Clusters_

_Debug is the only incant that takes no input and returns no output. Its sole use is to showcase the contents of the Akashic Stack for any Magus that needs to physically see what it contains._

**Example Spells:**

ETE DEBUG -> Allows the caster to see the Akashic and view their own cluster.

ETE GEA DEBUG -> Allows the caster to see the Akashic and view their own position.

**FANANIM**

**SYNTAX**

_Akashic -> No Clusters_

_Fananim does not need any clusters to function._ 

_Akashic <- Number (Arcana)_

_Fananim returns a Number Cluster, representing the amount of arcana a spell costs at the point where FANANIM can be found. This means that two FANANIMs may return two separate Numbers if they are in separate parts of a spell._

_Fananim is generally used to gauge how much a spell costs to ensure that it stays within a Magus's magical budget. Spell costs are dynamic, so there are times when this incant can be important to know\._

**Example Spells:**

ETE FANANIM DEBUG -> Allows the caster to see the Akashic and view the cost of using the ETE Incant.

ETE GEA FUS FANANIM DEBUG -> Allows the caster to see the Akashic and view the cost of this simple light spell.

**FANANIMVER**

**SYNTAX**

_Akashic -> Entity/Node_

_Fananimver requires an entity or node target._

_Akashic <- Number (Arcana)_

_Fananimver returns a Number Cluster, representing the amount of arcana a target has._

**Example Spells:**

ETE FANANIMVER DEBUG -> Allows the caster to see the Akashic and view the amount of arcana they have left.
