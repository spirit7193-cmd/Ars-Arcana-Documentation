Archiving Incants deal with one block, the Scroll Shelf. Additionally, there is only one Incant of this kind that exists.

In this section, you will learn its name and function.

**BURFANGEA**

**SYNTAX**

_Akashic -> Vector, Number_

_Burfangea requires 2 Clusters to function properly._

_The Vector Cluster serves as the position of the Scroll Shelf in the dimension._

_The Number Cluster serves as the slot to look in. As with most Akashic things, slots begin counting at 0. A Scroll Shelf has 6 of these "slots", implying values of 0-5._

_Akashic <- List/Null_

_Burfangea is the incant that represents "find spell in shelf". Provided with the location of a Scroll Shelf, it returns the Spell List in the specified slot. If there is nothing in the slot, a Null Cluster is returned instead. Whatever is returned is added to the top of the Akashic Stack._

**Example Spells:**

100 45 34 SINAFIL 0 BURFANGEA -> Returns the Spell List in a Scroll Shelf at the position \[100, 45, 34], otherwise returns Null if nothing is there. Remember, the provided Vector is arbitrary.
