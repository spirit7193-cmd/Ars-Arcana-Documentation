There is a Library that stands at the center of Nowhere. Its walls are sculpted sandstone; its hallways are enchanted papyrus. Not many ever find their way there, but hidden inside are a myriad of books. In this section, you will learn how to pry open the occult secrets of books. In fact, all Incants here pull from Written/Writable Books.

**KALATULGIM**

**SYNTAX**

_Akashic -> Frame Item/Item_

_Kalatulgim requires 1 Cluster to function properly._

_The Item/Frame Item Cluster must be a Book & Quill or Written Book._

_Akashic <- Word/Null/No Clusters_

_Kalatulgim is the incant that "gets the title of a book". If the book has a title, it will be returned. If it doesn't, Null is returned. Whatever is returned is added to the top of the Akashic Stack. If there is no Book, nothing is returned._

**KALATULSIN**

**SYNTAX**

_Akashic -> Frame Item/Item_

_Kalatulsin requires 1 Cluster to function properly._

_The Item/Frame Item Cluster must be a Book & Quill or Written Book._

_Akashic <- Word/Null/No Clusters_

_Kalatulsin is the incant that "gets the author of a book". If the book has an author, their name will be returned. If it doesn't, Null is returned. Whatever is returned is added to the top of the Akashic Stack. If there is no Book, nothing is returned._

**KALATULNIM**

**SYNTAX**

_Akashic -> Frame Item/Item_

_Kalatulnim requires 1 Cluster to function properly._

_The Item/Frame Item Cluster must be a Book & Quill or Written Book._

_Akashic <- Number/No Clusters_

_Kalatulnim is the incant that "gets the amount of pages in a book". The result is returned as a Number, including blank pages. This Number is added to the top of the Akashic Stack. If there is no Book, nothing is returned._

**KALATULBUR**

**SYNTAX**

_Akashic -> Frame Item/Item, Number_

_Kalatulnim requires 2 Clusters to function properly._

_The Item/Frame Item Cluster must be a Book & Quill or Written Book._

_The Number Cluster refers to the index of a Page in the Book. Anything above 49 is set to 49._

_Akashic <- Word/No Clusters_

_Kalatulbur is the incant that "gets the contents on a page in the book". The result is returned as a Word Cluster that is added to the top of the Akashic Stack. If there is no Book, nothing is returned. At this point, why would you NOT be using a book?!_

**KALATULYIS**

**SYNTAX**

_Akashic -> Item, Number, Word_

_Kalatulyis requires 3 Clusters to function properly._

_The Item Cluster must be a Book & Quill or Written Book._

_The Number Cluster refers to the index of a Page in the Book. Anything above 49 is set to 49._

_The Word Cluster refers to the text to put inside a specified Page in the Book. Words that are more than 256 characters long are cut to fit._

_Akashic <- No Clusters_

_Kalatulyis is the incant that writes to a Book. Because of this, there is no result. However, the change should be obvious in the text itself._

**KALYISAFAN**

**SYNTAX**

_Akashic -> Item, Number_

_Kalatulyis requires 2 Clusters to function properly._

_The Item Cluster must be a Book & Quill or Written Book._

_The Number Cluster refers to the index of a Page in the Book. Anything above 49 is set to 49._

_Akashic <- List_

_Kalyisafan is the incant that pulls from a Page in a Book and transforms it into a Spell List. This List is added to the top of the Akashic._

_As the only method of converting Words into Spell Lists, this Incant gained a reputation in the ancient world. Who can tell if this reputation will not bleed into the modern age?_
