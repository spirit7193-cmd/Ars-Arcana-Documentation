The Incants that manipulate blocks are few and far between. However, they are forever helpful, depending on use.

**HUSCOR (Break Earth)**

**SYNTAX**

_Akashic -> Vector_

_Huscor requires a location to function properly._

_Akashic <- No Clusters_

_Arcana Cost: 60_

_Huscor means "break earth", which is exactly what this does. It attempts to break a block with the power of a Stone Pickaxe, Axe, Shovel or Hoe. With Puissant HUSCOR, the Incant naturally grows stronger._

_HUSCOR always takes its cost. It doesn't matter if it cannot break the block at that very moment, as long as it can be broken eventually. However, if the block is truly unbreakable akashically, this used arcana is refunded._

**Example Spells:**

ETE GEA MALIBFIE SINVERA HUSCOR -> Attempts to break the block directly under the caster.

**SINCOR (Create Earth)**

**SYNTAX**

_Akashic -> Vector, Item_

_Sincor requires 2 Clusters to function properly._

_A Vector Cluster that determines where the block is going to be placed._

_An Item Cluster that determines what block is going to be placed based on the item itself._

_Akashic <- No Clusters_

_Arcana Cost: 20_

_Sincor means "create earth", which is a bit of a misnomer. It attempts to place a block at a position, based on the Item Cluster provided to it. The item is then appropriately consumed as well. It is something of a builder's tool in some cases._

**Example Spells:**

ETE BIEGEA MALIBIE SINVERA ETE 0 BURITE SINCOR -> Attempts to place the first item in the caster's inventory above their head.

**MALCOR (Move Earth)**

**SYNTAX**

_Akashic -> Vector, Vector_

_Malcor requires 2 Clusters to function properly._

_A Vector Cluster that determines the first position._

_A Vector  Cluster that determines the second position._

_Akashic <- No Clusters_

_Arcana Cost: 20_

_Malcor means "move earth", which it does a good job of doing. It directly swaps the positions of two valid blocks, so long as both are in loaded chunks of the dimension. In recent millennia, the swapping of blocks with air allows for solid shields._

**Example Spells:**

ETE BIEGEA MALIBIE SINVERA ETE GEA MALCOR -> Attempts to swap the block above the caster's head with the one below their feet.

**SINJAE (Create Redstone Energy)**

**SYNTAX**

_Akashic -> Vector_

_Sinjae requires a location to work properly._

_Akashic <- No Clusters_

_Arcana Cost: 150_

_Sinjae means "create redstone (energy)". It conjures a redstone source (with a power of 15) at the location provided, so long as the block is not occupied by another block._

**Example Spells:**

ETE GEA SINJAE -> Conjures a redstone signal at the caster's feet. It usually works because it is not often that the caster is in another block. Keyword: usually.

**KALASTE (Step Triggered Sigil)**

**SYNTAX**

_Akashic -> Vector, Vector, Number (Arcana), Item_

_Kalaste requires 4 Clusters to function properly._

_A Vector Cluster that determines the position to fire the sight ray from._

_A Vector Cluster that determines the direction the sight ray will go in. Whatever this hits is where the Sigil will be placed._

_A Number Cluster that determines the maximum amount of arcana the Sigil will hold._

_An Item Cluster that SHOULD hold a valid Spell Page._

_Akashic <- No Clusters_

_Arcana Cost: Same cost as the Number Cluster._

_Kalaste is an Incant that conjures a Sigil. This Sigil is triggered when an entity steps on it, drawing on its internal arcana to perform the spell saved to it._
