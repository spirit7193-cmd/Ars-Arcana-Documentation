As a magical language, arcanic must have ways to discern and manipulate dialogue. That is where Word Incants become useful. In fact, their versatility allows them to be more useful than magi think. There are even some Magi that communicate via akashic means, though only in the most dire of times.

**SINVERAWIS**

**SYNTAX**

_Akashic -> Word, Word_

_Sinverawis requires 2 Word Clusters to function properly._

_Akashic <- Word_

_Sinverawis is the incant that means "add/combine words and phrases". It attaches the second Word Cluster to the first one, effectively combining them. This new Word Cluster is added to the top of the Akashic Stack._

**Example Spells:**

/We are / /free!/ SINVERAWIS DEBUG -> Allows the caster to see the Akashic and view a Word Cluster that reads "We are free!".

**HUSAWIS**

**SYNTAX**

_Akashic -> Word, Word_

_Husawis requires 2 Word Clusters to function properly._

_Akashic <- Word_

_Husawis is the incant that means "seperate phrase by phrase". It uses the second Word Cluster as a separator , which splits the first one into pieces based on the second Word's absence. This new List of Word Clusters is added to the top of the Akashic Stack._

**Example Spells:**

/I + am + 8/ /+/ HUSAWIS DEBUG -> Allows the caster to see the Akashic and view a List of Word Clusters, containing "I ", " am ", and " 8".\`

**HUSASULWIS**

**SYNTAX**

_Akashic -> Word_

_Husasulwis requires 1 Word Cluster to function properly._

_Akashic <- List (Word)_

_Husasulwis is the incant that means "separate phrase by whitespace". It divides the Word Cluster provided by whitespaces (" "). This new List of Word Clusters is added to the top of the Akashic Stack._

**Example Spells:**

/I + am + 8/ HUSASULWIS DEBUG -> Allows the caster to see the Akashic and view a List of Word Clusters, containing "I", "+", "am", "+", and "8".\`

**WISANUM**

**SYNTAX**

_Akashic -> Word/Number_

_Wisanum requires a Word OR Number Cluster to function properly._

_Akashic <- Number/Word_

_Wisanum is the incant that seems to mean "conversion", specifically conversion from Word to Number and vice versa. Word Clusters are converted to their Number forms and Number Clusters are converted to their Word forms. Whatever is returned is added to the top of the Akashic Stack._

**Example Spells:**

/5/ WISANUM DEBUG -> Allows the caster to see the Akashic and view the Number 5.

5 WISANUM DEBUG -> Allows the caster to see the Akashic and view the Word “5”.

**IRASINWIS**

**SYNTAX**

_Akashic -> Word_

_Irasinwis requires 1 Word Cluster to function properly._

_Akashic <- No Clusters_

_Irasinwis the incant that is taken to mean "divine announcement". It takes the Word Cluster provided and announces it to the world through the Akashic. Naturally, it cannot be tracked by means available to magi. However, do note that this announcement affects the entire world; too many "divine" announcements in a short amount of time is bound to irk someone... or something._

**Example Spells:**

/I am fine!/ IRASINWIS -> Announces "I am fine!" to the entire World.

**SAHEJIT**

**SYNTAX**

_Akashic -> Entity, Word_

_Sahejit requires 2 Clusters to function properly._

_Akashic <- No Clusters_

_Sahejit is the incant that is taken to mean "message". It takes the Word Cluster provided and announces it to the entity, if they can receive it. Historically, this has been used for a variety of "communication" spells._

**Example Spells:**

ETE ESIJIT /You are next./ SAHEJIT -> Sends a message saying "You are next." to the entity (player) the caster is looking at.

**BURVIRSAH**

**SYNTAX**

_Akashic -> Entity_

_Burvirsah requires 1 Cluster to function properly._

_Akashic <- Word/Null_

_Burvirsah is the incant that is taken to mean "get last message". It extracts the last message sent from the entity provided and returns it as a Word Cluster. If they have no last chat message, it returns Null. Whatever is returned is then added to the top of the Akashic Stack._

_Bote: This incant does not function on players without the supplementary add-on "Ars Additionals", which is experimental. There are, however, things that may say things that cannot normally be heard…_

**Example Spells:**

ETE SINJIT ESIJIT BURVIRSAH SAHEJIT -> Sends the caster the last message the entity they are looking at sent. In certain situations, it can glean messages from other entities that cannot send messages.
